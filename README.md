
# Spam Email Classifier Using Machine Learning

This project detects whether a message is spam or ham using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- TF-IDF Vectorizer
- Naive Bayes Algorithm

## Accuracy
96.59%

## Features
- Text preprocessing
- Spam detection
- Message prediction
- High accuracy classification
